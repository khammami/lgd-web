# Local Guides Directory (LGD) — Web

Website for the Local Guides Directory app — a directory for Google Local Guides to share their profiles and connect with fellow explorers.

**Live:** [lgd.khammami.tn](https://lgd.khammami.tn)

## Tech Stack

- [Astro](https://astro.build/) 5 (static output)
- [Tailwind CSS](https://tailwindcss.com/) 4
- [Firebase](https://firebase.google.com/)
- React 19 (client island for analytics)

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
